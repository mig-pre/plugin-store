# one-click-token-launch
One-click multi-launchpad token creation with bundled buy, IPFS metadata upload, and MEV protection across 6 launchpads on Solana and BSC — all driven by OnchainOS Agentic Wallet with TEE signing.

## Highlights
- 6 launchpad coverage: pump.fun, LetsBonk, Bags.fm, Moonit, Four.meme, Flap.sh
- Bundled initial buy: deploy token + snipe your own launch in one atomic transaction
- IPFS metadata: auto-upload image + description to Pinata or pump.fun IPFS
- MEV protection: Jito bundle submission for Solana launches
- Configurable supply, decimals, initial liquidity, and dev allocation
- Supports Solana and BSC chains
- All execution via OnchainOS Agentic Wallet (TEE signing, no API key needed)
- Explorer links auto-generated post-launch (Solscan, BscScan)
