# rust-sol-price
## Overview
A Rust CLI tool that queries real-time SOL price data from the OKX public API.
Core operations:
- Query current SOL price in USD
- Display 24-hour price change percentage
- Fetch trading volume data
Tags: `solana` `price` `rust` `analytics`
## Prerequisites
- No IP restrictions
- Supported chain: Solana
- Supported token: SOL
- onchainos CLI installed
## Quick Start
1. Run `rust-sol-price` to get the current SOL/USDT price with 24h change.
2. Alternative: `onchainos market price --token SOL --chain solana`.
