# mainstream-spot-order
Multi-chain DEX spot trading system with a 6-signal ensemble (Momentum, EMA, RSI, MACD, Bollinger Bands, BTC Overlay) on 15-minute bars across SOL, ETH, BTC, BNB, AVAX, and DOGE — with AI-driven auto-research strategy optimization, per-pair backtesting, and paper/live trading modes.

## Highlights
- 6 mainstream token pairs: SOL/USDC, ETH/USDC, BTC/USDC, BNB/USDC, AVAX/USDC, DOGE/USDC
- 6-signal ensemble on 15-minute candles: Momentum, EMA crossover, RSI, MACD, Bollinger Bands, BTC Overlay
- AI-powered auto-research: automatically optimizes strategy parameters per pair
- Per-pair data collection, backtesting, and performance tracking
- Paper mode default — live mode requires explicit confirmation
- Multi-chain: Solana, Ethereum, BSC, Avalanche
- All execution via OnchainOS Agentic Wallet (TEE signing, no API key needed)
- Web dashboard at localhost:3250
