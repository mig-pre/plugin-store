# go-price-demo
A minimal Go demo plugin that queries cryptocurrency prices from the OKX public API for CI testing purposes.

## Highlights
- Query real-time ETH/BTC prices from OKX public API
- Support for multiple token symbols (ETH, BTC, SOL)
- Read-only operation with no wallet access required
- No API keys needed - uses public endpoints only
- Returns comprehensive price data including 24h statistics
- JSON output format for easy integration
- Built in Go for performance and reliability
- Designed for CI testing and demonstration
